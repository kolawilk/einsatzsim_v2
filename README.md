# 🚒 Einsatzsim v2

> Making fire department play sessions more atmospheric and fun for parents and kids.

## Vision

Einsatzsim transforms ordinary Playmobil fire station play into immersive emergency missions. Through a carefully crafted state machine, atmospheric audio layers, and full-screen visuals, it creates the perfect backdrop for imaginative play between parents and children.

Built with love for iPad and iPhone, designed to stay on during play sessions, and crafted to be toddler-friendly with big buttons, emojis, and fire department red.

## Features

- 🎮 **State Machine**: Guided mission flow from call to return
- 🔊 **Layered Audio**: Ambient sounds, effects, and sequences
- 🖼️ **Full-Screen Visuals**: Immersive scene images
- ⚙️ **Semi-Automatic Mode**: Configurable auto-transitions
- 📱 **AlwaysOn**: Stays awake during play
- 🎯 **Mission System**: YAML-defined missions, no repeats per session

## Tech Stack

- Next.js 14+ with App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Howler.js for audio

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open on iPad at http://localhost:3000
```
